# Support Ticket System

A production-grade React + TypeScript application demonstrating Priority Scheduling with Aging Algorithm for customer support ticket management.

## Features

- 🎫 **Ticket Management** - Create, track, and resolve support tickets
- ⚡ **Priority Scheduling** - Automatic ticket prioritization (High → Medium → Low)
- ⏰ **Aging Algorithm** - Prevents ticket starvation by auto-upgrading priorities
- 📊 **Real-time Analytics** - Live stats and execution logs
- 💾 **Persistent Storage** - Local storage for data persistence
- 🎨 **Modern UI** - Tailwind CSS with smooth animations

## Tech Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion
- date-fns
- React Hook Form + Zod

## Quick Start

### Prerequisites
- Node.js 18+
- pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/support-ticket-system.git

# Navigate to project
cd support-ticket-system

# Install dependencies
pnpm install

# Start development server
pnpm dev
